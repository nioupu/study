熟悉创建用户命令
sudo useradd -m *                 -m：自动为新建用户创建家目录
sudo usermod -aG sudo *                   usermod：修改已有用户的属性      a：append 追加        G：groups 组          给用户sudo权限
sudo gpasswd -d * sudo 将用户*从sudo组移除
groups *

删除用户：sudo userdel -r *    -r:彻底删除

熟悉查看目录命令
ls
ls -l         显示详细信息
ls -a        显示所有文件，包括以.开头的隐藏文件


修改文件/目录的访问权限：chmod （控制谁能读，写，执行），解决文件无法打开/修改常见问题
所有者 u 所属组 g 其他用户 o
读r=4  写w=2 执行x=1
数字法：chmod 644 test.txt 对应-rw-r--r--
字母法：chmod u+x script.sh


修改文件/目录的所有者/所属组:chown
sudo chown * test.txt (更改所有者)
sudo chown * : sudo test.txt(更改所有者，并修改所属组)

touch：创建空文件
touch test.txt
cat > test.txt   (写入命令覆盖原有内容)
cat >> test.txt(追加在末尾命令)
nano test.txt（像记事本一样编辑）
